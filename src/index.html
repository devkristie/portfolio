Gazetteer Web Mapping App

View Website (opens in a new tab) View Source Code (opens in a new tab)

Notice:
Source code is private due to course requirements, available on request.

Gazetteer Web Mapping App This website was created as part of my IT Career Switch Coding Traineeship and is one of the final three projects I needed to complete to pass the course. The project is a mobile-first website that operates equally well on desktop. It provides detailed country profiles, presenting demographic, climate, geographical, and other relevant data such as population, area, capital city, currency, time zone, and local news.

The task was to adapt a CodePen template and implement full functionality. I used HTML, CSS, Bootstrap, JavaScript, jQuery, and AJAX, along with PHP cURL, hosted on a PHP-enabled web server using Apache. The project also integrates multiple third-party APIs, including OpenCage for geocoding, OpenWeather for weather forecasts, GeoNames for country and Wikipedia data, and Open Exchange Rates for currency conversion. Additionally, Leaflet and its plugins – Leaflet.markercluster, Leaflet.easyButton, and Leaflet.extraMarker – were used to create interactive, clusterable maps with custom markers.

Through this project, I gained experience in integrating multiple APIs, handling asynchronous requests, dynamically updating the frontend, implementing PHP backend functionality, and working with interactive mapping libraries.

Full-stack Developer Portfolio This website was created as part of my IT Career Switch Coding Traineeship and was one of the final three projects required to complete the course. The task was to select a Bootstrap template and transform it into a personalised portfolio website by updating all text and imagery, removing unneeded sections, and adding a fully functioning contact form built with PHP. The form sends messages directly from the website to my email address.

I used HTML, CSS, Bootstrap, JavaScript, and PHP to build this project, focusing on understanding how templates are structured and how to customise them effectively. I also learned how to handle form submissions securely in PHP and connect the front end interface with a simple but functional back-end process.

Through this project, I gained practical experience in Bootstrap layout customisation, template editing, and implementing PHP-based form handling. It strengthened my confidence in working with both client-side and server-side code.

Company Directory App This website was created as part of my IT Career Switch Coding Traineeship and is one of the final three projects required to pass the course. The task was to build a Company Directory web application using a provided CodePen layout as the design reference and replicate its structure as closely as possible.

The application was built using HTML5, CSS3, JavaScript, PHP, and MySQL. It includes full CRUD functionality, allowing users to create, view, edit, and delete employee records stored in a MySQL database. I implemented dynamic features such as form validation, responsive layout adjustments, and live record updates using AJAX to improve user experience. The backend was built with PHP, handling all data operations securely and communicating with the MySQL database.

This project strengthened my understanding of full stack development, including database design, server-side programming, and integrating frontend and backend functionality into one complete application.
